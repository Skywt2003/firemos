# Firemos

Send memo to your [Memos](https://usememos.com) instance in Firefox. Still WIP.

## Instruction

After installation, set your Memos API in options.

Press `Ctrl + Shift + M` (on macOS, `Cmd + Shift + M`) to open the popup.
